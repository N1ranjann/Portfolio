01 — Overview

Legacy hardware,
production soul.

BlackBox is a high-availability, self-hosted server node repurposed from a Lenovo IdeaPad x86 laptop. Running Ubuntu Server with Docker and CasaOS, it hosts a growing stack of self-managed services in a zero-trust network topology.

LVM (Logical Volume Manager) underpins the storage layer, giving flexible and resizable volume management for persistent container data — no data loss on container updates or recreation.

"Maximum utility from minimum hardware. Every service earns its uptime."

The full configuration — compose files, network schemas, LVM setup, and troubleshooting notes — is documented in the GitHub repository for reproducibility and learning.

02 — Architecture

Zero-trust
by design.

Tailscale provides a WireGuard-based mesh VPN overlay — every device that needs to reach BlackBox must be enrolled and authenticated. There is no open port exposed to the public internet.

AdGuard Home acts as the network-level DNS resolver and ad blocker for the entire home network, running in a privileged container with direct access to port 53. It intercepts all DNS queries before they leave the network.

Uptime Kuma provides real-time monitoring and alerting for every service, ensuring that any container downtime is detected and flagged within seconds — even from a mobile device over Tailscale.

03 — Capabilities

What
BlackBox
runs.

  • 01

    Container Orchestration — Docker Compose manages all services declaratively. Every service is reproducible, versionable, and restartable with a single command.

  • 02

    LVM Storage Management — Logical Volume Manager provides flexible, resizable storage pools with snapshot support for safe container data persistence.

  • 03

    Zero-Trust Networking — Tailscale WireGuard mesh ensures no service is reachable without device authentication. The attack surface is zero public-facing ports.

  • 04

    Network-Level Privacy — AdGuard Home intercepts all DNS at the gateway, blocking trackers and ads for every device on the network before queries escape.

  • 05

    Self-Hosted Media — Immich delivers a full-featured private photo library with ML-powered search, face recognition, and album management — no cloud required.

  • 06

    Comprehensive Documentation — Compose templates, network architecture schemas, LVM walkthroughs, and troubleshooting logs all live in the public GitHub repository.

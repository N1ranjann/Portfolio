01 — Overview

Security that
thinks ahead.

Logix is a high-density, real-time Security Information and Event Management (SIEM) platform designed for modern security operations. It combines the power of Behavioral Detection, SIGMA Rule Correlation, and Stateful Incident Management into a single, unified engine.

Built on Flask with SQLite for rapid deployability and Docker for containerised operations, Logix is engineered to be self-contained, portable, and operationally ready without heavyweight dependencies.

"Security operations need signal, not noise. Logix is built around precision detection over volume alerts."

MITRE ATT&CK alignment ensures every detection maps to a known adversary technique — giving security teams context, not just events.

02 — Architecture

Three engines,
one platform.

Logix is built around three core modules: the Behavioral Detection engine monitors real-time event streams for anomalous patterns that deviate from established baselines. The SIGMA Rule Correlator maps incoming log data against a library of structured detection rules.

The Stateful Incident Manager tracks the lifecycle of every detected event — from initial trigger through triage, assignment, and resolution — with full audit trails. All three modules feed into a unified dashboard, providing a single operational view.

03 — Capabilities

What Logix
detects.

  • 01

    Behavioral Detection Engine — Monitors real-time event streams for anomalous activity, establishing dynamic baselines and flagging deviations that match known attack patterns.

  • 02

    SIGMA Rule Correlation — Applies a curated library of community and custom SIGMA rules to incoming log data, providing structured, vendor-agnostic detection logic.

  • 03

    Stateful Incident Management — Tracks every security event through its full lifecycle — creation, triage, assignment, and resolution — with complete audit trails.

  • 04

    MITRE ATT&CK Mapping — Every detection is automatically mapped to a specific ATT&CK technique and tactic, giving analysts immediate adversary context.

  • 05

    Real-Time Dashboard — A unified operational view aggregating alerts, incidents, and metrics across all detection modules in a single high-density interface.

  • 06

    Docker-Ready Deployment — Fully containerised for one-command deployment, with no external service dependencies — runs anywhere Docker runs.
